# Portfolio project: "KeksoBooking"

[GitHub Pages Link](https://naumch1k.github.io/1692143-keksobooking-23/)

## Description
"KeksoBooking" is a portfolio project created as a part of a study at [HTML Academy](https://htmlacademy.ru/ "Adaptive Website Coding and Automation"), marketplace for users to list, discover and book properties in the city of Tokyo.

## Functionality
* Filter and sort properties per options chosen
* Set address field value by moving marker across the map
* Form validation
